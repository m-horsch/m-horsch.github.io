**Solvers already presented:**
  * IDS (simple, enhanced)
  * Look-up table (LUT), Distance Table (DIST)
  * Greedy search using various distance estimate heuristics
  * A\* using various distance estimates


**Solvers implemented but not yet discussed**
  * Iterative deepening A\*
  * Composition of partial solvers.  This is a very broad category.  Also very cool, I think.
    * TLC solving
    * Centre Solving
    * Solver based on partitioning (large puzzles)

**Solvers not yet implemented**
  * Weighted A\*
  * Anytime solvers, using weight progression and Weighted A\*
  * Solvers specialized for puzzles with exchangeable tiles.
  * Meta-heuristic solvers  
  * Solvers based on permutations rather than operations.
